# Neural Networks
TP2 for Artificial Intelligence Systems course from I.T.B.A.

## Pre-requisites
Add the `src` folder & subfolders to the Matlab/Octave path.

## Execute
You can use any of the perceptrons's scripts to solve problems they are
 supposed to solve (or, and, xor, parity, symmetry). You can choose
 any unit function package you want or add a new one (recommended folder:
 `activation_function`; just try to follow the other folders structure).
